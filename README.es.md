# 🎓 Student Dropout Prediction – Flask Web App

## 📌 Descripción del Proyecto
Este proyecto consiste en el desarrollo de un modelo de Machine Learning para predecir la **deserción estudiantil** (Dropout) utilizando un dataset sintético de 10,000 estudiantes.

La variable objetivo es:

- **Dropout**
  - `0` → El estudiante continúa estudiando
  - `1` → El estudiante abandona

Se desarrolló un modelo de clasificación y posteriormente se integró en una aplicación web usando **Flask**. Finalmente, la aplicación fue desplegada en **Render**.

---

## 📊 Dataset

El dataset incluye variables demográficas, académicas y de comportamiento, tales como:

- Edad
- Género
- Ingresos familiares
- Horas de estudio
- Asistencia
- GPA / CGPA
- Nivel de estrés
- Departamento académico
- Entre otras

El dataset contiene valores faltantes (~5%) y variables categóricas que fueron procesadas antes del entrenamiento.
